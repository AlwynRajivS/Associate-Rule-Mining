# 🎓 Associate Rule Mining — Advanced Data Mining Platform

An interactive **web-based data mining platform** for performing **Association Rule Mining, Market Basket Analysis, Multilevel & Multidimensional Analysis**, with **AI-powered insights**.

---

## ✨ Features

* 📂 Upload Excel/CSV datasets (auto column detection)
* 🔗 Association Rule Mining (Apriori Algorithm)
* 🛒 Market Basket Analysis (transaction-based)
* ⚖️ Quantitative Analysis (distribution & ranges)
* 🌳 Multilevel Analysis (hierarchical data exploration)
* 🔷 Multidimensional Analysis (cross-tab heatmaps)
* 🤖 AI Insights using Groq API
* 📊 Interactive charts (Chart.js)
* 🌗 Dark / Light Mode UI

---

## 🧠 Technologies Used

* HTML, CSS, JavaScript
* Chart.js (Visualization)
* XLSX.js (Excel Parsing)
* Groq API (AI Insights)

---

## 🚀 How to Use

1. Open the application in browser
2. Upload dataset (.xlsx / .csv)
3. Explore:

   * Quantitative Analysis
   * Multilevel Hierarchy
   * Multidimensional Heatmap
   * Association Rules
4. (Optional) Add Groq API key for AI insights

---

## 📊 Core Concept

The platform uses the **Apriori Algorithm** to discover:

* Frequent Itemsets
* Strong Association Rules
* Support, Confidence, Lift metrics
